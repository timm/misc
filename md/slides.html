Slide 1 title

Some super quickly created demo slides. The syntax is simple and effective for creating slides on the fly.

  • No special tools needed:
    • Use basic markdown to create slides.
    • Start slide titles with # and use bullet points for content.
  • Advanced Options:
    • Add custom HTML tags like <center> for alignment.
    • Use LaTeX commands in Pandoc Beamer format (e.g., \\center, \\large).

The slide syntax is so simple that you can quickly create a handful of slides on basically any device in any editor. E.g., on your mobile on the way to the meeting where you need the slides. Right before the meeting starts you use pandoc to create the actual slides from your source.
